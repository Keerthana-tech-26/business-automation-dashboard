CSV Format Instructions

Your CSV file must have these columns (in this exact order):

  • invoice_number - Unique invoice number (e.g., INV-2024-001)
  • client_name - Client/company name
  • client_email - Client email address
  • amount - Amount in numbers (e.g., 50000.00)
  • status - One of: DRAFT, SENT, PAID, OVERDUE
  • issue_date - Date in YYYY-MM-DD format
  • due_date - Date in YYYY-MM-DD format
  • description - Invoice description
Important: Column names must match exactly (lowercase).
Example CSV Content:
invoice_number,client_name,client_email,amount,status,issue_date,due_date,description
INV-2024-004,ABC Corporation,contact@abc.com,125000.00,SENT,2024-10-25,2024-11-25,Website development services
