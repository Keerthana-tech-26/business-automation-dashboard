CSV Format Instructions

Your CSV file must have these columns (in this exact order):

  • title - Expense title/name
  • amount - Amount in numbers (e.g., 5000.00)
  • category - One of: OFFICE, TRAVEL, UTILITIES, SALARY, MARKETING, OTHER
  • date - Date in YYYY-MM-DD format (e.g., 2024-10-25)
  • description - Description (optional)
Important: Column names must match exactly (lowercase).
Example CSV Content:
title,amount,category,date,description
Office Chairs,15000.00,OFFICE,2024-10-25,Ergonomic chairs for team
Flight Tickets,8500.00,TRAVEL,2024-10-24,Mumbai business trip
